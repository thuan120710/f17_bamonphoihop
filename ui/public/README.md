1. **5count.mp3** - Âm thanh countdown 5 giây
2. **rightchose.mp3** - Âm thanh qua checkpoint

## Copy từ gameracing:

```bash
# Windows
copy ..\..\f17_gameracing\html\sounds\5count.mp3 .
copy ..\..\f17_gameracing\html\sounds\rightchose.mp3 .
```

Sau đó chạy `npm run build` để Vite tự động copy vào `html/sounds/`
